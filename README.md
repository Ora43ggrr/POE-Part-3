Claim Management System
A web application for managing lecturer claims and approvals.

Features
Lecturer Portal

Submit new claims

Upload supporting documents

Track claim status in real-time

Coordinator Portal

Review and verify claims

Approve or reject submissions

Filter and search claims

Real-time status updates

Technologies Used
ASP.NET Core MVC

Entity Framework Core

SQL Server

HTML/CSS/JavaScript

Bootstrap

Installation
Clone the repository

Update database connection string in appsettings.json

Run database migrations:

bash
dotnet ef database update
Build and run the application:

bash
dotnet run
Usage
Lecturers can log in to submit claims and documents

Coordinators can log in to review and approve claims

Both roles can track claim status in real-time
